Display the count of custom posts in the WordPress dashboard in the 'At a glance' widget, like posts, pages and comment count.

== Description ==
This plugin helps you to see the post count for your custom post in the 'At a glance' widget in the WordPress dashboard. It also has a setting page where you can choose the post types for which you wish to show the count.

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Custom Post Count At A Glance"
3. **Install and Activate** Custom Post Count At A Glance from your Plugins page
4. **Click** on the new menu item "CPT Glance" under "Settings" and select the custom post to display count!
